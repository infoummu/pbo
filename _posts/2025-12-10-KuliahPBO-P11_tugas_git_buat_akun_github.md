## 1. Pengantar: Mengapa Git dan GitHub Penting dalam PBO?

Pemrograman Berorientasi Objek (PBO) adalah tentang membangun perangkat lunak yang terstruktur, modular, dan dapat dipertahankan. Untuk proyek PBO yang kompleks dan dikerjakan dalam tim, penggunaan **Version Control System (VCS)** adalah wajib.

* **Git** berfungsi sebagai **mesin waktu** yang mencatat dan mengelola setiap perubahan kode Anda, memungkinkan Anda bekerja secara paralel pada fitur yang berbeda dan kembali ke versi stabil kapan saja.
* **GitHub** adalah *platform* berbasis web yang menyediakan **repositori sentral** untuk kolaborasi kode.

Menguasai Git dan GitHub mengajarkan **disiplin kerja tim**, **manajemen risiko kode**, dan **alur kerja kolaborasi** yang menjadi standar industri, keterampilan esensial dalam pengembangan perangkat lunak berbasis objek skala nyata.

## 2. Tugas: Buatlah Akun di GitHub

**Setiap mahasiswa diwajibkan untuk membuat akun GitHub pribadi dan mendokumentasikan prosesnya secara lengkap dalam format PDF.**

### A. Langkah-Langkah Teknis

1.  Akses situs web **GitHub** di **`https://github.com/`**, atau clik **[github](https://github.com/){:target="_blank"}**.
2.  Lakukan proses pendaftaran akun baru (*Sign Up*) menggunakan *email* aktif Anda.
3.  Pilih *username* yang profesional dan mudah diidentifikasi (disarankan menggunakan nama asli atau kombinasinya, cth: `nama-anda-npm`).
4.  Lengkapi semua langkah verifikasi hingga akun Anda berhasil dibuat dan Anda berada di *dashboard* utama.
5.  **Ambil *screenshot*** untuk setiap langkah penting (pendaftaran, verifikasi *email*, dan *dashboard* akhir).
6.  Pada akun yang sudah jadi, jelaskan secara singkat di bagian mana Anda akan melihat *repository* yang nantinya akan digunakan untuk tugas PBO.

### B. Dokumentasi Laporan (Format PDF)

**Dokumentasikan seluruh proses di atas dalam bentuk berkas PDF dengan struktur berikut:**

* **Halaman 1 (Halaman Profil WAJIB):**
    * Nama Lengkap:
    * Nomor Pokok Mahasiswa (NPM):
    * Mata Kuliah: Pemrograman Berorientasi Objek (PBO)
    * Dosen Pengampu: (Isi Nama Dosen Anda)
* **Halaman Selanjutnya (Isi Laporan):**
    * Judul: Dokumentasi Pembuatan Akun GitHub.
    * Sertakan *screenshot* yang jelas untuk setiap langkah, diikuti dengan deskripsi singkat yang menjelaskan aksi yang dilakukan pada langkah tersebut.
    * Pastikan *screenshot* menampilkan *username* GitHub yang Anda buat.
