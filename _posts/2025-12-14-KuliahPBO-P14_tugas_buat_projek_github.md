### Tugas Membuat Projek baru, 
1. Semua Mahasiswa Info3 dan Info4 Membuat Tugas Projek dengan tema bebas 
2. Buat Projek Besar menggunakan bahasa pemrograman python menggunakan skill hasil belajar anda dari awal pertemuan MK PBO.
3. Buat Projeknya di Github dan Tulis deskripsi tentang projeknya dan dokomentasi tentang penggunaan codenya/perintahnya di file README.md
4. Dalam mengerjakan projek, anda bisa menggunakan **Kata kunci** dibawah untuk mencari panduan di internet: 
    - **"Cara buat projek baru atau repositori baru di github"**, silahkan cari di internet dengan kata kunci ini atau semisalnya, atau
    - **"Cara clone projek baru dan push ke github"**, lihat atau cari di internet dengan kata kunci ini.
5. Kumpul LINK projeknya (contoh: https://github.com/infoummu/projek1) ke LINK sekalian Absen di https://pbo.tifor.info/absenandlink.php
6. Sudah hrus ada di github/kumpul  hari Selasa (16/12/2025)
