## MK PBO FINAL Projek (UAS): 
Mahasiswa Semester 3 yang mengambil MK PBO diminta membuat sebuah Final Projek (UAS dalam bentuk projek) yang dikerjakan secara bersama terdiri dari maksimal 5 orang minimal 3 orang, detailnya sebagai berikut:
1. Buat Tugas Projek Akhir Kalaborasi dengan teman2 di Github
2. Tugas Projek Kelompok, maksimal 5 orang dan minimal 3, (segera langsung buat kelompoknya)
3. Bagi Tugas dalam kelompok agar Terdiri dari:
    - 1 orang MAINTAINER (ketua Kelompok), 
    - 1 atau 2 orang Bertanggung Jawab membuat Dokumentasi (di file README.md)
    - Dokumentasi/Panduan/HowTo terdiri dari:
        - Nama Projek
        - Nama2 Kontributor
        - Deskripsi Projek
        - Yang Dibutuhkan Oleh projek 
        - How to use/Bagaimana Menggunakan Projek ini 
        - ...
        - Contoh:  [https://github.com/infoummu/pbo_projek_uas](https://github.com/infoummu/pbo_projek_uas){:target="_blank"}
    - Teman2 lainnya sebagai kontributor (mengerjakan projek)
4. Maintainer bertugas Buat Projek, dan teman2 lainnya melakukan "FORK" projeknya 
    - Lakukan kontribusi dengan menambahkan code ke projek sesuai arahan atau kesepakatan 
    - Lakukan "Pull Reques" ke projek utama 
    - Maintener melakukan "Merger" 
    * Kata Kunci : fork, pull request,
    * Cari di Internet dengna kata kunci:  
        - untuk ketua kelompok (MAINTAINER): "Bagaimana membuat repository/project baru di github online"
        - untuk anggota/kontributor: "How to fork and make a contribution to github project online"
        - untuk anggota/kontributor: "Bagaimana melakukan fork dan berkontribusi ke projek di github secara online"
5. Maintainer mengarahkan semua kontributor (devops junior/beginer)
6. Sudah harus kumpul di : 
    - https://pbo.tifor.info/absenandlink.php
    - waktu : MAINTAINER Projek sudah HARUS Mengumpulkan LINK Projek pada Hari Sabtu Malam 
7. DiPresentasikan Offline/Online
